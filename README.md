# 红利助手

一个面向 A 股 / 港股的移动端股息追踪工具，帮助投资者管理持仓、对比股息率、计算税后收益。

## 功能

- **发现** — 按行业浏览股票，支持隐藏、内联编辑静态数据
- **自选** — 管理持仓（股数、成本价、税务类型），实时获取最新价格
- **收益** — 聚合持仓盈亏及股息收益一览
- **矩阵** — 股息率对比图表（基于 Recharts）
- **设置** — 汇率配置、数据备份/恢复

## 技术栈

React 18 · TypeScript · Vite · Tailwind CSS · Zustand · Recharts

## 快速开始

```bash
npm install
npm run dev      # 启动开发服务器（含 API 代理）
npm run build    # 类型检查 + 生产构建
npm run preview  # 本地预览生产构建
```

## 数据来源

实时行情通过腾讯财经接口（`qt.gtimg.cn`）获取，开发环境由 Vite 代理，生产环境需自行配置反向代理将 `/api/*` 转发至对应域名。

## 税务说明

- **A 股**：红利免税
- **港股（港户）**：预扣税 10%
- **港股（H 股沪深通）**：预扣税 20%
- **港股（非 H 股沪深通）**：预扣税 28%
